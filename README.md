# Workcity Chat WordPress Plugin

This plugin integrates the **Workcity Chat system** into WordPress.  
It adds a shortcode `[workcity_chat]` that displays a real-time chat box.

---

## 🚀 Features
- WordPress shortcode `[workcity_chat]`
- Chat box with real-time messaging
- Connects to backend Socket.IO server
- Simple, responsive UI

---

## ⚙️ Requirements
- WordPress 6.x
- PHP 7.4+
- Workcity Chat backend running at:
http://localhost:4000



## 📂 Installation
1. Copy plugin to:

wp-content/plugins/workcity-chat



2. Activate plugin in WordPress Dashboard.
3. Create a page and insert shortcode:
[workcity_chat]


4. Open the page → chat is ready 🎉.


## 🛠️ Technologies
- PHP (WordPress plugin)
- JavaScript (Socket.IO client)
- Node.js backend
